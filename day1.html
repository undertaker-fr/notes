第 1 章

什么是"终端"?

你平常用电脑大多用的是图形界面(双击图标、点鼠标、看窗口)。但电脑还有另一种用法,叫终端(Terminal)——一个黑底白字的窗口,你在里面用打字的方式给电脑下命令。

Windows 自带的终端有两种:

  • cmd(老派,叫"命令提示符")
  • PowerShell(新派,功能更强,我们用这个)

怎么打开 PowerShell

  1. 按键盘 Win + R
  2. 弹出小框,输入 powershell
  3. 按回车 → 蓝色或黑色窗口弹出
💡 程序员每天用终端,因为打字比点鼠标快,而且能做很多图形界面做不到的事(比如自动化、批量处理)。
第 2 章

"路径"是什么?

路径就是文件或文件夹的"完整地址"。打个比方:

盘符(C:\、D:\)楼栋号码
文件夹楼层和房间
文件房间里的具体物品
路径从楼栋到物品的完整地址

举个例子:

C:\Users\Administrator\Desktop\照片.jpg

读法:去 C 这栋楼 → 进 Users 这层 → 进 Administrator 房间 → 进 Desktop 子房间 → 找到 照片.jpg 这个文件。

两种路径写法

类型例子啥时候用
绝对路径C:\Users\Administrator\Desktop从盘符开头,在哪儿都能用
相对路径Desktop"我当前位置旁边的 Desktop",必须当前位置正好有这个文件夹
🤔 思考题:你现在在 C:\Users\Administrator,旁边有个文件夹叫 Desktop。下面哪条命令能进 Desktop?
点开看答案
两条都可以:
① cd Desktop(相对路径,因为 Desktop 就在你旁边)
② cd C:\Users\Administrator\Desktop(绝对路径,完整地址)
第 3 章

核心 5 条命令

1. pwd — 我现在在哪

print working directory(打印当前工作目录)。打开 PowerShell 第一件事就该问"我在哪"。

PS C:\Users\Administrator> pwd
Path
----
C:\Users\Administrator

2. dir — 周围有啥

directory(目录)。列出当前文件夹里的所有内容。

PS C:\Users\Administrator> dir
Mode    LastWriteTime     Length  Name
----    -------------     ------  ----
d-----  2026/4/26 23:21           Desktop
d-----  2026/3/9   0:59           Documents
-a----  2026/4/27  0:06   20832  .claude.json

读懂这个输出:

  • Mode:d----- = 文件夹(directory),-a---- = 文件
  • LastWriteTime:最后修改时间
  • Length:文件大小(字节);文件夹这一栏是空的,因为文件夹本身不占空间
  • Name:名字

3. cd 路径 — 去哪

change directory(切换目录)。后面跟的可以是相对路径或绝对路径。

PS C:\Users\Administrator> cd Desktop
PS C:\Users\Administrator\Desktop>

注意提示符前面的路径变了——这是你"进了 Desktop 房间"的证据。

4. cd .. — 退一步

两个英文点 .. 代表"上一级"。cd .. 就是"我要回去"。

PS C:\Users\Administrator\Desktop> cd ..
PS C:\Users\Administrator>

5. mkdir 名字 — 新建文件夹

make directory。一行命令凭空建出一个文件夹,跟图形界面里"右键-新建文件夹"是一回事,但更快。

PS C:\Users\Administrator\Desktop> mkdir my-folder
PS C:\Users\Administrator\Desktop> dir
... 可以看到 my-folder 出现了
第 4 章

实用神技巧

① 拖放文件夹自动填路径

这一招超神:

  1. 在 PowerShell 里输 cd (末尾带个空格)
  2. 不要按回车
  3. 用鼠标把任意一个文件夹从资源管理器拖进 PowerShell 窗口
  4. 它会自动把完整路径填进去
  5. 按回车

避免手打长路径,一秒到位。

② Tab 键自动补全

打到一半懒得继续,按一下 Tab 键:

  • 输 cd Des → 按 Tab → 自动变成 cd Desktop
  • 多个匹配项时,继续按 Tab 在它们之间循环

③ 大小写问题

Windows 终端不区分大小写。cd desktop 和 cd Desktop 都能进。

⚠️ 警告:Linux 服务器(以后部署项目会接触)严格区分大小写,Desktop 和 desktop 是两个不同的东西。所以现在就养成"原文件夹怎么写就怎么写"的习惯,以免以后踩坑。

🏆 解锁的能力

  • 能从键盘启动 PowerShell 终端
  • 看一眼提示符就知道自己当前在哪个文件夹
  • 能用绝对路径和相对路径自由进出文件夹
  • 能列出当前文件夹的所有内容,识别哪些是文件夹哪些是文件
  • 能新建文件夹
  • 会用 Tab 自动补全和拖放路径两个效率神技
